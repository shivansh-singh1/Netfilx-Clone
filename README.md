# Netflix Clone

A static Netflix landing page clone built using HTML and CSS. This project focuses on recreating the visual layout, responsiveness, and overall UI of Netflix’s homepage without using JavaScript or any backend logic.

---

## Features

* Netflix-style landing page UI
* Responsive design for desktop and mobile screens
* Navigation bar with language selector and sign-out button
* Hero section with call-to-action
* Trending section with horizontal scrolling layout
* Feature highlights section
* FAQ section with expandable questions (CSS-based)
* Footer with multiple informational links

---

## Tech Stack

* HTML5
* CSS3

## How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/Netflix-Clone.git
   ```
2. Open the project folder
3. Open `index.html` in your browser

No additional setup or dependencies are required.
